Personal Portfolio Website

Welcome to my personal portfolio website, designed and developed from scratch to showcase my skills, projects, and experience in Frontend Development and UI/UX Design.


Overview

I created this portfolio to represent my professional identity as a budding software developer. The goal was to build a clean, responsive, and interactive website that highlights my work and makes it easy for visitors to learn more about me.


Features

Responsive Design – Works smoothly on all devices (desktop, tablet, mobile)
Dark / Light Mode Toggle – Better accessibility and user preference
Project Filtering & Detailed Views – Showcasing my work clearly
Animated Elements – Subtle UI animations for a modern feel
About Me & Resume Section – Quick snapshot of my background
Contact Form with Social Links – Easy way to reach out


Tech Stack

Frontend: HTML5, CSS3, JavaScript
Frameworks / Libraries: React.js, TailwindCSS
Design Tools: Figma (for layout planning and UI ideas)
